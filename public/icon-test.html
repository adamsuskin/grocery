How to Generate Icons

Recommended Method: Use the Sharp-based generation script
# Install Sharp (if not already installed)
npm install --save-dev sharp

# Generate all PNG icons
node scripts/generate-icons.js

Alternative methods:

# Using Inkscape (command line)
inkscape public/icons/icon-template.svg \
  --export-type=png \
  --export-filename=public/icons/icon-192x192.png \
  -w 192 -h 192

# Using ImageMagick
convert -background none -resize 192x192 \
  public/icons/icon-template.svg \
  public/icons/icon-192x192.png

Testing Checklist

Visual Tests

  • All standard icons load without errors
  • Maskable icons load without errors
  • Icons are sharp and clear at all sizes
  • Icons are visible on all background colors
  • 16x16 icon is recognizable
  • Favicon appears in browser tab

Technical Tests

  • Open DevTools → Application → Manifest
  • Verify all icons are listed in manifest
  • Click each icon to preview in DevTools
  • Test maskable icons at Maskable.app
  • Install PWA and verify home screen icon
  • Check splash screen on mobile

Cross-Platform Tests

  • Test on Android Chrome (install PWA)
  • Test on iOS Safari (add to home screen)
  • Test on desktop Chrome (install app)
  • Test on desktop Edge (install app)
  • Verify Windows tile icon

Quick Actions

After generating icons:

  1. Reload this page to see updated icons
  2. Clear browser cache if icons don't update
  3. Create/update manifest.json
  4. Update index.html with icon references
  5. Test PWA installation on mobile device
